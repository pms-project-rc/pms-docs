# Glosario del Dominio (Ubiquitous Language)

Este documento define los términos fundamentales del negocio para asegurar que desarrolladores y stakeholders hablen el mismo idioma.

| Término | Definición |
| :--- | :--- |
| **Vehículo** | Unidad física identificada por placa (puede tener mensualidad o convenio). |
| **Servicio de Lavado** | Proceso de limpieza asociado a un vehículo, con tipo de lavado y estados (En espera, En proceso, Terminado). |
| **Tipo de Lavado** | Categoría con precio base y minutos de parqueo gratuito incluidos (ej. Sencillo, General, Especial). |
| **Registro de Parqueo** | Entrada/salida de vehículo, con cálculo de tarifa y posibles exenciones. |
| **Mensualidad** | Plan fijo de parqueo por periodo (30 días) vinculado a un vehículo. |
| **Convenio** | Acuerdo empresarial que otorga descuentos a vehículos afiliados (Flotas). |
| **Flota** | Conjunto de vehículos asociados a un convenio específico. |
| **Turno** | Intervalo operativo gestionado por un administrador (ingresos/gastos/servicios). |
| **Gasto** | Salida de dinero categorizada que afecta rendimiento. |
| **Vale** | Anticipo o préstamo al empleado que se descuenta automáticamente de sus bonos. |
| **Bono** | Incentivo calculado como porcentaje del valor de lavados realizados por un lavador. |
| **Rendimiento Operativo** | Métrica financiera clave: `Ingresos – (Gastos + Bonos)`. |
| **Ocupación** | Número de vehículos presentes simultáneamente en parqueadero. |
| **Tiempo Gratuito** | Minutos de parqueo sin costo otorgados por tipo de lavado. |
| **Exención** | Acción manual auditada que marca un cobro como no aplicable. |
